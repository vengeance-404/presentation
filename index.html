Abstract

  • System Overview: A Flask-based web application leveraging supervised machine learning for real-time phishing URL detection with 93.6% accuracy.
  • Problem Domain: Addresses escalating phishing threats by classifying URLs as benign or malicious using 12 extracted features.
  • Key Components: Advanced feature engineering from URL attributes, model inference via REST API, and intuitive user interface.
  • Technical Stack: Employs Scikit-learn, Pandas, and Flask for a scalable, production-ready cybersecurity ML pipeline.

Machine Learning Model

Algorithm Selection

Gradient Boosting Classifier (Scikit-learn) selected for sequential error reduction and superior performance on imbalanced datasets.

Training Process

3-fold Cross-validation with GridSearchCV for hyperparameter optimization. Training set: 80%, Test set: 20%.

Model Serialization

Used Pickle and Dill libraries for secure model serialization, ensuring version compatibility and deployment stability.
